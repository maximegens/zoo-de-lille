# ---------- Application Android Zoo de Lille ---------- #

> **Application mobile Android du parc zoologique de Lille**

![zoodelille](https://bytebucket.org/maximegens/zoo-de-lille/raw/bbb497443f4dc9c30130b410100bf4494e9039dc/Data/maquette_situation.png?token=368af6f760d22a20ca379be24aa0d28bb8d72c24)


**Développeur** : 

* Maxime Gens
* Matthieu Poudroux ?

**Objectif** : Développer une application Android pour le Zoo de Lille, ceci afin de visiter le parc de façon moderne et ludique.

Fonctionnalités principales :

* Indication de l'ouverture du parc en temps réel
* Consultation d'actualités concernant le zoo
* Consultation de la listes des animaux
* Recherche d'un animal
* Agenda des activités pédagogique
* Jeux Interactif
* Notification d’événements
* Plan d'accès 
* Plan interactif
* Contacter le parc
